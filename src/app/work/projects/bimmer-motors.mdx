## Overview

Migration and redesign of a WordPress website to a new host, while preserving SEO performance.

## Key Features

- **Modern Responsive Design**: Using a modern WordPress Block Theme, Elementor plugin, and custom JavaScript and CSS.
- **Custom Media Production**: Lead creative direction for on-site photography, videography and graphic design.
- **Integrated Customer Reviews**: Installed and customized a WordPress plugin to merge and displays any reviews from Google, Facebook and Yelp in the single feed.
- **SEO Retention**: Ensured that the sitemap, url slugs, and HTML structure and content for all blog pages on the new site were identical to the old. HTML structure and content of the home page and other pages were preserved to greatest extent possible as well. 
- **Google Maps Navigation**: "Get Directions" button using url query parameters that trigger google maps navigation, in app or browser, to the repair shop from the user's current location.


## Technologies Used

- **WordPress**: Modern Block Theme. Elementor plugin.
- **Front-End Technologies**: JavaScript, CSS, HTML.
- **FileZilla**: Migration of WP Content files using FTP.
- **All-in-One SEO (AIOSEO)**: Benchmarking and validation of SEO performance retention.
- **Content Delivery Network**: For faster media asset load times.
- **URL query parameters**
- **Photoshop**: Media editing and production.
